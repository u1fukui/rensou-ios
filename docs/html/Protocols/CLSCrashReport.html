CLSCrashReport Protocol Reference

Conforms to NSObject
Declared in Crashlytics.h

Overview

The CLSCrashReport protocol exposes methods that you can call on crash report objects passed to delegate methods. If you want these values or the entire object to stay in memory retain them or copy them.
